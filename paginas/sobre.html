Sobre a Calculadora de IMC

Uma ferramenta desenvolvida para promover a conscientização sobre saúde e bem-estar

Nossa Missão

Nosso objetivo é fornecer uma ferramenta acessível e fácil de usar para que qualquer pessoa possa calcular seu Índice de Massa Corporal (IMC) e entender melhor sua saúde. Acreditamos que informação é o primeiro passo para hábitos mais saudáveis.

Queremos democratizar o acesso a informações básicas de saúde, ajudando as pessoas a monitorarem seu peso de forma consciente e responsável.

Como Surgiu Este Projeto

Este projeto foi desenvolvido como uma iniciativa educacional para demonstrar o uso de tecnologias web (HTML, CSS e JavaScript) na criação de ferramentas úteis para o dia a dia.

Percebemos que muitas pessoas têm dúvidas sobre seu peso ideal, mas nem sempre têm acesso fácil a informações confiáveis. Decidimos então criar uma calculadora online que fosse:

  • Totalmente gratuita
  • Fácil de usar
  • Baseada em padrões científicos
  • Acessível em qualquer dispositivo
  • Com informações educativas

Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando apenas tecnologias front-end padrão da web:

HTML5
CSS3
JavaScript (ES6+)
Font Awesome Icons
Google Fonts
Design Responsivo

O código foi estruturado de forma modular para facilitar a manutenção e expansão do projeto. Todas as páginas são estáticas e não requerem backend, podendo ser hospedadas em qualquer servidor web simples.

Aviso Importante

Esta calculadora de IMC é uma ferramenta informativa e educacional. Os resultados fornecidos são baseados na fórmula padrão do IMC e nas classificações da Organização Mundial da Saúde.

Este site não substitui a consulta com um profissional de saúde. O IMC é apenas um dos muitos indicadores de saúde e deve ser interpretado por um médico ou nutricionista qualificado, que considerará outros fatores como composição corporal, histórico médico, idade, sexo e estilo de vida.

Se você tem preocupações sobre seu peso ou saúde, consulte um profissional de saúde qualificado.
